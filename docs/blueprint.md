# **App Name**: Registro Académico NIU

## Core Features:

- Extracción de Datos: Extraer datos estructurados del archivo de Excel cargado, analizando la información del estudiante, los detalles del curso, las calificaciones y la información de los créditos.
- Visualización del Registro Académico: Mostrar los datos extraídos en un formato tabular limpio, similar a la hoja de Excel original, asegurando que todos los detalles, como el nombre del curso, la calificación, los créditos y la fecha de finalización, sean visibles.
- Cálculo del GPA: Calcular automáticamente el promedio de calificaciones (GPA) en función de las calificaciones y los créditos extraídos de la hoja de Excel. Esta función requiere un análisis preciso de los detalles del curso.
- Validación de Datos y Verificación de Errores: Implementar comprobaciones de validación para garantizar la precisión e integridad de los datos extraídos, resaltando cualquier discrepancia o información faltante para su revisión.

## Style Guidelines:

- Color primario: Azul marino oscuro (#1A237E), reflejando una sensación académica tradicional.
- Color de fondo: Gris claro (#EEEEEE), para un entorno de lectura limpio y sin distracciones.
- Color de acento: Oro apagado (#BDBDBD), para resaltar la información clave sin ser demasiado brillante.
- Fuente del encabezado: 'PT Sans', sans-serif. Fuente del cuerpo: 'Inter', sans-serif. Nota: actualmente solo se admiten fuentes de Google.
- Usar íconos simples y profesionales para la navegación y la presentación de datos (por ejemplo, birrete de graduación para la información del estudiante, libros para los cursos).
- Mantener un diseño limpio y organizado, similar a la hoja de Excel pero optimizado para la presentación web.
- Animaciones sutiles al cargar datos o cambiar entre secciones para proporcionar transiciones suaves.
